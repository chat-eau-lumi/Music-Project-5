Overall Description:

Player that will generate 4 measures of melody based on carefully selected probabilites as well as rhythms and then uses a track system to play the measures as well as decide when to generate new melodies and rhythms.
Users can also input rhythm and melody tracks as well as edit some basic features of each track.

gen_fn.srp

osc_interaction.srp
MidRange: code to create harmonies
